# Weather
#### Приложение показывающее прогноз погоды. В мобильной версии скролл реализован по типу тач скрин(вправо или влево) 
[Project link/URL](https://lowerrider.github.io/Weather/)

## Technology stack
____

+ React
+ SASS
+ RESTful API
+ AXIOS
+ React-multi-carousel
+ Material-ui/core
+ Адаптивная верстка
+ Webpack
____
